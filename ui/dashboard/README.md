# Network Agent User Interface

User interface for Google Cloud Network Agent demo with the following features: 

* Dynamic Network Topology
* Chat interface with hierarchy of network agents
* Log Viewer
* Performance Metric Dashboard for all running network functions

## Deploy the Network Agent Dashboard to GCP

The agent backend and dashboard are deployed by running the following command.

```
install.sh -n
```

## Running the Network Agent Dashboard locally ( from VSCode )

To run the network agent on your local machine

* [Install flutter SDK](https://docs.flutter.dev/get-started)
* [Install flutter VSCode flutter extensions](https://docs.flutter.dev/tools/vs-code)
